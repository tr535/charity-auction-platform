## Installation & Setup

### Backend (Server)
1. Navigate to project-web-api folder.
2. Update Connection String in appsettings.json.
3. Run:
cd project-web-api
dotnet ef database update
dotnet run

### Frontend (Client)
1. Navigate to angular folder.
2. Run:
cd angular
npm install
ng serve

Open http://localhost:4200 in your browser.
