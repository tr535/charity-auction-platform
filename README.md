Installation & Setup

Backend (Server)

Navigate to project-web-api folder.

Update Connection String in appsettings.json.

Run these commands:
cd project-web-api
dotnet ef database update
dotnet run

Frontend (Client)

Navigate to angular folder.

Run these commands:
cd angular
npm install
ng serve
Open http://localhost:4200 in your browser.
